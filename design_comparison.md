# Design Comparison - Original vs New

## Оригинальный сайт (contcatlog-ktp5mmht.manus.space)

### Каталог
- Логотип: "Каталог контейнеров 365" - синий текст с иконкой контейнера
- Фон: темно-синий (#0f172a или похожий)
- Карточки контейнеров:
  - Фото занимает верхнюю часть карточки
  - Бейдж состояния в правом верхнем углу фото
  - Название контейнера белым жирным шрифтом
  - "Размер" и "ID" серым текстом
  - Значения размера и ID белым текстом
  - Цена желтым/золотым цветом (70 000.00 ₽)
  - Кнопка "Смотреть" с синей рамкой
- Фильтры: выпадающие списки "Размер" и "Состояние"
- Поиск: поле с иконкой лупы

### Различия с новым сайтом:
1. ✅ Темный фон - соответствует
2. ✅ Карточки контейнеров - структура соответствует
3. ✅ Бейджи состояния - соответствуют
4. ⚠️ Цена - нужно сделать желтым цветом
5. ⚠️ Кнопка "Смотреть" - нужно сделать с синей рамкой вместо заливки
6. ✅ Фильтры и поиск - соответствуют
7. ⚠️ Фото в карточках каталога - не отображаются (нужно проверить)

## Действия для исправления:
1. Изменить цвет цены на желтый/золотой
2. Изменить стиль кнопки "Смотреть" на outline с синей рамкой
3. Проверить отображение фото в карточках каталога
